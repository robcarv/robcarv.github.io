Docker Services

Pi 1 Γאפ Primary (8GB)

  • ΓאóPi-hole (DNS ad-blocker)
  • ΓאóUnbound (recursive DNS resolver)
  • ΓאóNginx Proxy Manager (reverse proxy)
  • ΓאóHomepage (dashboard)
  • ΓאóUptime Kuma (monitoring)
  • ΓאóWatchtower (auto-updates)
  • ΓאóDozzle (container logs)

Pi 2 Γאפ Media & Automation (4GB)

  • ΓאóJellyfin (media server)
  • ΓאóSonarr (TV series manager)
  • ΓאóRadarr (movie manager)
  • ΓאóProwlarr (indexer manager)
  • ΓאóqBittorrent (torrent client)
  • ΓאóHome Assistant (smart home)
  • ΓאóZigbee2MQTT (IoT bridge)

Pi 3 Γאפ Developer Tools (4GB)

  • ΓאóGitea (Git server)
  • ΓאóJenkins (CI/CD)
  • ΓאóPortainer (container management)
  • ΓאóMinIO (S3-compatible object storage)
  • ΓאóN8n (workflow automation)
  • ΓאóVaultwarden (password manager)
  • ΓאóSyncthing (file sync)
